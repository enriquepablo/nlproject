Time expressions¶

There are 2 kinds of time expressions in nl: instants, and durations (intervals of time). They just basically boil down to integers and pairs of integers, except for some special treatment of the “now” concept.

Instants¶

An instant is really simply a positive natural number. To create an instant, you import Instant from nl, and instantiate it with an integer.

>>> from nl import Instant
>>> i1 = Instant(1)

As is the case with predicates, you don’t tell instants; that would be rather pointless. You use them to build facts that use verbs that are instantaneous. Appart from instantiating them with integers, you can also instantiate them with the special string “now”, and that will be the same as instantiating them with the number of seconds since “the epoch”.

>>> from time import time
>>> now = Instant('now')
>>> now2 == int(time())
True

Durations¶

A duration is composed of 2 instants, where the first instant is a smaller integer than the second. To create a duration, you import Duration from nl, and implement it with an instant named start and another instant named end.

>>> from nl import Duration
>>> d1 = Duration(start=Instant(1), end=Instant(2))
