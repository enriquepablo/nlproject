Welcome to nlproject¶

These docs are work in progress

nlproject is a set of software facilities that provide development environments for logic programming in Python. At its core is nl, a bare bones logic system, comprising the basic building blocks for sentences and rules, and an in-memory knowledge base to hold these sentences and rules. On top of it is nlserv, that provides persistence for nl‘s knowledge bases and an XML-RPC interface to them.

Support¶

There is a mailing list for nlproject at google groups. You can also open an issue in the tracker.

To install the software¶

The software is hosted at github. To install it, clone the repository, make a python2.5 virtualenv (or use your system’s python2.5), and execute the buildout:

$ git clone git@github.com:enriquepablo/nlproject.git
$ cd nlproject
$ virtualenv --no-site-packages --python=python2.5 .
$ source bin/activate
$ python bootstrap.py
$ bin/buildout
